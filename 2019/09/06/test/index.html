使用Hexo 和 GitHub 搭建博客

Sep 6, 2019

为什么写博客

在网路快餐，微博等应用横行的今天，不得不说我变的越来浮躁了，变得越来越懒，学习的主观能动性都要没了，也很少静下来集中思考一些东西，越来越少的记录自己的想法。有时突然想写点东西，但总感觉到力有不逮。
其实，之前也有时不时写blog记录的习惯，但是仅限在CSDN和简书上记录，也曾想过自己购买服务器搭建一个，但是还是每搁住自己太懒了，动手能力太次了，不仅没搭建起来，把CSDN索性也抛弃了。近期，与朋友/同事 聊天 深感中年危机的同时，也意思到了和优秀的人差距越来越大。遂决定痛改前非，先从改变“懒做起”，捡起曾经的Bolg记录，以便督促自己不断的进步，也在希望往后的日子里，翻阅自己曾经写的东西时，会收到意想不到的喜悦。
遂决定痛改前非，不在被自己的“懒”耽误，开始用博客记录自己的奇思妙想，学习，生活，但愿以后翻阅时能有不一样的感受。
在此，特别感谢同事吴大佬，在构建Blog时不遗余力的帮忙。

使用GitHub博客

Github有Github Pages这个伟大的功能，每一个账号可以建一个yourname.github.io的站点，可以使用Jekyll/Hexo模板创建。对于不懂技术的人来说，可能会有一点折腾，但是绝对不难，折腾完后就能收获满满的成就感了，然后就可以安安静静地写博客啦。
简单说来，用Hexo建立Github博客分为以下几个步骤：

  • 安装Hexo
  • 配置GitHub仓库
  • 修改模版主题，直到满意为止（Hexo提供了好多供选择的主题）
  • 将修改后的所有文件上传到yourname.github.io中
    参考链接，不在赘述了，如果在搭建过程中碰到问题，可以进行Google,几乎全部可以搜到解决方案。

最后

从开始有这个想法到现在写完这篇博客，其实断断续续经历了有一段时间（不是因为难，只是因为我的拖拉，嗯，想尝试的朋友千万不要被这句话给吓到）。目前Bolg里面没有任何内容，但这是刚开始，我们默默的期待着。。。
